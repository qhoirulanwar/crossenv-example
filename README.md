### crossenv-example

Menggunakan cross-env untuk mengelola environment (development mode/production mode) pada project nodejs

```sh
$ git clone https://github.com/qhoirulanwar/crossenv-example.git
$ cd crossenv-example
$ npm install
```

### run development mode
```sh
$ npm run start
```
or
```sh
$ npm run dev
```

### run production mode
```sh
$ npm run prod
```
